# 🎵 Music Mosh MAX v3.0 - Unified Edition

**Production-ready music discovery with procedural 3D visuals**

## 🚀 Quick Deploy

### Deploy to Vercel (Recommended)

1. Push this code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" → Import your repo
4. Click "Deploy" (auto-detects Next.js)
5. Done! You'll get a `*.vercel.app` URL

## ✨ Features

- **Procedural 3D Visuals** - Three.js scenes that change with mood/genre/theme
- **Real YouTube Discovery** - Finds actual tracks via Invidious API + curated fallbacks
- **Chaos Mode** - Auto-cycling visuals every 8 seconds
- **Social Sharing** - Share buttons for Twitter, Facebook, Reddit, WhatsApp
- **Error Resilience** - Graceful degradation on all devices
- **Mobile Optimized** - GPU-aware performance scaling

## 🛠️ Tech Stack

- Next.js 16 (App Router)
- Three.js r134 (3D graphics)
- Framer Motion (animations)
- Tailwind CSS v4
- TypeScript

## 📦 Local Development

```bash
bun install
bun run dev
```

Open http://localhost:3000

## 🎮 How to Use

1. Select Mood + Genre (Theme optional)
2. Click "Find My Track"
3. Get a real YouTube video
4. Toggle "Chaos Mode" for endless visuals
5. Click "Share" to share with friends
